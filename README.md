# 🎱 双色球选号助手

基于历史开奖数据的双色球号码分析工具，包含：
- 📊 冷热号分析
- 🔮 智能选号（多种策略）
- 📈 走势图表
- 🔄 在线数据更新（500.com）

## 使用

纯静态网页，直接打开 `index.html` 即可。

## 数据源

- 内置历史数据（兜底）
- 在线更新：[500.com 双色球历史](https://datachart.500.com/ssq/history/history.shtml)

## 技术

- 纯 HTML / CSS / JavaScript
- 零依赖，零后端
- 部署在 GitHub Pages

🌐 在线访问：https://cdszeyao.github.io/ssq-tool/
